# C++ IM 网关演进式学习项目

## 📖 项目概述

一个通过 **八个阶段演进式实现** 以**数据驱动**的高性能IM网关学习项目，旨在深入掌握C++网络编程、内存管理与并发优化的核心原理。

### 🎯 核心学习理念
- **自主思考**：自主设计项目，**非网课项目**，有决策日志，压测报告，性能报告，架构图，请查看[./docs/](./docs/)（目前处于阶段一，只有决策日志有内容）
- **踩坑演进**：从最简单的阻塞IO开始，逐步演进到epoll多路复用、内存优化、零拷贝和多线程架构，每一步都亲身体会“为什么要有下一步”
- **数据驱动**：每个阶段都有明确具体真实的性能指标对比和问题发现。**工具链**：wrk2、perf、Valgrind、火焰图
- **深度优先**：拒绝黑盒调用，直面底层原理，具体学习过程均在[./study_draft/](./study_draft/)
